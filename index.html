Neuroevolutionary Procedural Level Generator — Personal Project

2026 — Genetic algorithm + learned fitness function for procedural 2D platformer levels

  • Built a genetic algorithm (tournament selection, column crossover, mutation, elitism) that evolves 40×12 platformer level genomes in Python, Pygame & NumPy
  • Designed a hand-coded fitness function scoring reachability (BFS with jump-arc abstraction), gap fairness, enemy density/placement, platform structure & verticality, and jump comfort (widest-path search)
  • Trained a from-scratch NumPy MLP (10 → 16 → 8 → 1, no PyTorch dependency) on a labeled dataset to predict a level's "fun" score, used as a learned, blendable drop-in replacement for the rule-based fitness function
  • Built a genome repair() pass enforcing playability invariants (guaranteed reachability, hazard pacing, gap-width caps, grounded enemies) verified as a stable, idempotent projection via property-based testing (Hypothesis) across hundreds of randomized genomes

03

Research Intern — VIT Vellore

2025 — Quantum Mixed-State Attention Network (QMSAN) for multilingual text classification

  • Designed QMSAN — a hybrid quantum-classical architecture integrating XLM-RoBERTa with parameterized quantum circuits (VQC, QRC) for attention computation via qubit-space expectation values
  • Implemented 4 model variants (Control, VQC, QRC, Hybrid) evaluated on XGLUE NC across English, German, Spanish, French, and Russian — Hybrid achieved 87% EN accuracy and ~80% on DE/ES, with avg 78% across all 5 languages
  • QRC variant was fastest to train (~51s/batch) while the Hybrid led on macro F1 (0.65) — demonstrating distinct accuracy-vs-efficiency trade-offs across quantum approaches
  • Used PennyLane and Qiskit for quantum simulation on standard GPU hardware (RTX 3050), showing hybrid QML is viable without real quantum devices
